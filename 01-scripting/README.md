# Giai đoạn 1 — Scripting & Tự động hóa

🎯 DevOps = tự động hóa. Bạn cần viết script để khỏi làm tay lặp đi lặp lại.

## Thứ tự học
1. `bash` — Script shell: biến, vòng lặp, if, hàm, xử lý file/log
2. `python` — Ngôn ngữ "dao đa năng" của DevOps: gọi API, xử lý JSON/YAML, automation

## ✅ Checklist
- [ ] Viết script bash backup file & hẹn giờ chạy bằng cron
- [ ] Viết script Python gọi một REST API và in kết quả
- [ ] Đọc/ghi file YAML & JSON bằng Python
- [ ] Hiểu exit code, stdout/stderr, pipe

## 🛠️ Dự án nhỏ
Script tự động: kiểm tra một website có "sống" không (HTTP 200), nếu lỗi thì ghi log + gửi cảnh báo.
