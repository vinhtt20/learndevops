# Giai đoạn 4 — Cloud

🎯 Chọn MỘT cloud để học sâu (đừng dàn trải). AWS phổ biến nhất → khuyên học AWS trước.

## Lựa chọn
1. `aws` — Phổ biến nhất, nhiều việc nhất → ưu tiên
2. `azure` — Mạnh ở doanh nghiệp dùng Microsoft
3. `gcp` — Mạnh về data/Kubernetes

## ✅ Checklist (lấy AWS làm ví dụ)
- [ ] Tạo tài khoản free tier, bật cảnh báo chi phí (RẤT QUAN TRỌNG kẻo bị tính tiền)
- [ ] Tạo EC2 (máy ảo), SSH vào, cài web server
- [ ] Hiểu: VPC, subnet, security group, IAM
- [ ] Lưu file lên S3, dùng RDS (database)
- [ ] Hiểu mô hình chi phí & cách tránh "cháy túi"

## 🛠️ Dự án nhỏ
Deploy app web (đã đóng gói Docker ở GĐ2) lên một EC2, mở được qua internet.
