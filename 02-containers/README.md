# Giai đoạn 2 — Containers (Docker)

🎯 Container là trái tim của DevOps hiện đại. "Chạy được trên máy tôi" → chạy được mọi nơi.

## Thứ tự học
1. `docker` — Image vs container, Dockerfile, volume, network, registry
2. `docker-compose` — Chạy nhiều container cùng lúc (app + database)

## ✅ Checklist
- [ ] Viết Dockerfile đóng gói một app (Node/Python) thành image
- [ ] Hiểu khác nhau: image, container, layer, volume
- [ ] Dùng docker-compose chạy app + database cùng lúc
- [ ] Đẩy image lên Docker Hub

## 🛠️ Dự án nhỏ
Đóng gói một app web + database PostgreSQL bằng docker-compose, chạy bằng 1 lệnh `docker compose up`.
