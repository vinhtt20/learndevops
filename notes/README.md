# 📓 Ghi chú học tập

- `daily-log/` — Mỗi ngày 1 file (vd: `2026-06-26.md`): hôm nay học gì, mắc gì, giải quyết sao.
  Thói quen này cực kỳ giá trị: vừa ôn lại, vừa là tư liệu viết blog/CV.
- `cheatsheets/` — Tổng hợp lệnh hay quên (docker, kubectl, git, linux...).
- `interview-prep/` — Câu hỏi phỏng vấn DevOps + câu trả lời của bạn.

> Mẹo: dạy lại người khác (hoặc viết blog) là cách học nhanh nhất. "Học bằng cách dạy."
