# Giai đoạn 0 — Nền tảng (Foundations)

🎯 **Quan trọng nhất với người trái ngành.** DevOps xây trên nền Linux + mạng + dòng lệnh + Git.
Vững phần này thì mọi thứ sau dễ; bỏ qua sẽ "lạc" liên tục.

## Thứ tự học
1. `01-linux` — File, quyền, process, package, dịch vụ (systemd)
2. `02-networking` — IP, port, DNS, HTTP/HTTPS, firewall, SSH
3. `03-command-line-bash` — Thành thạo terminal: pipe, grep, find, vim cơ bản
4. `04-git-github` — Branch, commit, pull request

## ✅ Checklist
- [ ] Tự cài & dùng máy Linux (VM VirtualBox / WSL2 / VPS rẻ)
- [ ] Dùng `ls cd cp mv rm chmod chown ps top grep find` không cần tra
- [ ] Giải thích: IP vs port vs DNS; HTTP request đi qua những gì
- [ ] SSH vào server từ xa bằng key (không password)
- [ ] Tạo repo, commit, push, mở pull request

## 🛠️ Dự án nhỏ
Thuê VPS rẻ (hoặc VM), SSH vào, cài nginx, đẩy 1 trang HTML lên, mở qua IP công khai.
