# 🎓 Dự án Capstone — Tổng hợp

Mục tiêu: chứng minh bạn làm được toàn bộ vòng đời DevOps. Đây là dự án "đinh" trong CV.

## Yêu cầu
Lấy một ứng dụng thật (vd: app web có frontend + backend + database) và:

1. **Code** → quản lý trên GitHub
2. **Container** → đóng gói Docker (GĐ2)
3. **CI/CD** → pipeline tự build/test/deploy (GĐ3)
4. **Cloud** → chạy trên AWS/Azure/GCP (GĐ4)
5. **IaC** → toàn bộ hạ tầng bằng Terraform (GĐ5)
6. **Kubernetes** → deploy lên K8s, scale, không downtime (GĐ6)
7. **Observability** → có dashboard + alert (GĐ7)
8. **Security** → scan image, quản lý secret an toàn (GĐ8)

## Sản phẩm cuối
- Repo GitHub công khai, README đẹp, có sơ đồ kiến trúc
- Bài viết blog/LinkedIn kể lại quá trình → gây ấn tượng nhà tuyển dụng
